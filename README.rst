====================================
Redshift database backend for Django
====================================

This product is tested with:

* python-2.7.x
* django-1.7.6, 1.7.10, 1.8.8


Differences from postgres_psycopg2 backend
==========================================

* Not use RETURNING.
* Not use SELECT FOR UPDATE.
* Not use SET TIME ZONE.
